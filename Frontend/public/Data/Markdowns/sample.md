#+ なぜ作ろうと思ったのか？(Rendalizeの利点) bg_color=none align=center
\NormalCard width=100% bg_color=none
?[File bg_color=#61baff content=**資料作りってめんどくさい**]
ハッカソンに出てプロダクトを作ったら
- Readme
- プレゼン資料
- Web(就活用のポートフォリオとして)
?[TrendingUp bg_color=#fd7979 content=**Webだからアニメーションの自由度が高い**]
Webには**Hover**や**Cursor**という武器がある(canvaやpptxではできないよね？)
?[Sparkles bg_color=#79fd8d content=**楽にオリジナル性の高い資料が作れる**]
テンプレートが**コンポーネント単位**なので組み合わせ次第でオリジナル性あふれるものが作れる
もちろん**コンポーネントの追加**もOK
